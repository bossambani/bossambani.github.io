Layoffs Data Cleaning Project

SQL | Data Cleaning

Comprehensive cleaning and standardization of a global layoffs dataset:

  • Duplicate Removal: Identified and eliminated duplicate records using ROW_NUMBER() and CTEs
  • Data Standardization: Normalized company names, industries (e.g., consolidated "Crypto" categories), and country formats
  • Date Conversion: Transformed text dates to DATE format using STR_TO_DATE
  • Null Handling: Implemented JOIN operations to fill missing industry data from related records
-- Example of duplicate handling
                            WITH duplicates_cte AS (
                            SELECT *,
                            ROW_NUMBER() OVER(
                                PARTITION BY company, location, industry, 
                                total_laid_off, percentage_laid_off, `date`,
                                stage, country, funds_raised_millions
                            ) AS row_num
                            FROM layoff_staging
                            )
                            DELETE FROM duplicates_cte WHERE row_num > 1;
